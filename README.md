# QUORIDOR

## Informations générales
Ce projet Quoridor est une implémentation en Java du jeu de stratégie. 
Le jeu implique le déplacement de votre pion vers l'autre côté du plateau tout en plaçant des murs pour bloquer le chemin de votre adversaire. 
L'application propose une interface graphique ainsi qu'une version en mode terminal.

## Installation
1. Ouvrez le dossier dans une IDE telle que Visual Studio Code ou IntelliJ avec JDK 21.
2. Naviguez vers le dossier `src`.

### Exécution du jeu avec l'interface graphique
- Compilez le jeu en utilisant la classe `Graphique`.

### Exécution du jeu sur le terminal
- Compilez le jeu en utilisant la classe `Jeu`.
